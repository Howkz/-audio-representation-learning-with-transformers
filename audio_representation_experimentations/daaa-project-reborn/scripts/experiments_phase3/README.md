# Launchers Phase 3

Ce dossier contient les launchers dédiés à la campagne corrective phase 3.

Scripts :

- `run_single_phase3_experiment.sh`
- `run_P3D01.sh`
- `run_P3B01.sh`
- `run_P3B02.sh`

```bash
bash scripts/experiments_phase3/run_P3D01.sh
bash scripts/experiments_phase3/run_P3B01.sh --resume
bash scripts/experiments_phase3/run_single_phase3_experiment.sh P3B02 --materialize-data
```
