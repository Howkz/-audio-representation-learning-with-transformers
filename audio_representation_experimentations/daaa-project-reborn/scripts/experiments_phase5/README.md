# Launchers Phase 5

Ce dossier contient les launchers dédiés à la campagne de test d'apprenabilité
phase 5.

Scripts :

- `run_single_phase5_experiment.sh`
- `run_P5D01.sh`
- `run_P5B01.sh`
- `run_P5B02.sh`

Usage :

```bash
bash scripts/experiments_phase5/run_P5D01.sh
bash scripts/experiments_phase5/run_P5B01.sh --resume
bash scripts/experiments_phase5/run_single_phase5_experiment.sh P5B02 --materialize-data
```
