# Launchers Phase 4

Ce dossier contient les launchers dédiés à la campagne corrective phase 4.

Scripts :

- `run_single_phase4_experiment.sh`
- `run_P4D01.sh`
- `run_P4B01.sh`
- `run_P4B02.sh`

Usage :

```bash
bash scripts/experiments_phase4/run_P4D01.sh
bash scripts/experiments_phase4/run_P4B01.sh --resume
bash scripts/experiments_phase4/run_single_phase4_experiment.sh P4B02 --materialize-data
```
